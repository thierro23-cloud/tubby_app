Extracción automática de PDFs de obras

Esta herramienta analiza documentos administrativos para extraer información relevante. El sistema busca el CSV en varias fuentes:

  • Metadatos del PDF
  • Texto interno del documento
  • Zona inferior del documento (donde suele aparecer el CSV)
  • OCR en caso de documentos escaneados

1. Subir nuevo PDF

Seleccione un documento administrativo en formato PDF. El sistema procesará automáticamente el archivo y extraerá los datos disponibles.

2. Proceso de análisis del documento

Cuando se sube un PDF, el sistema realiza automáticamente los siguientes pasos:

  1. Lectura completa del documento para extraer el texto disponible.
  2. Búsqueda del CSV (Código Seguro de Verificación) para garantizar que el documento no ha sido procesado anteriormente.
  3. Generación de un objeto JSON con el texto extraído y los datos detectados en el documento.
  4. Almacenamiento del resultado en una tabla intermedia (tbl_obras_pdf) que posteriormente será utilizada para alimentar la tabla final de control de obras.

4. Objetivo del sistema

El objetivo principal de esta herramienta es automatizar la gestión de documentos administrativos relacionados con obras y ocupaciones de vía pública.

La extracción automática permite reducir errores humanos, evitar duplicados y facilitar la integración de los datos en el sistema de control de obras municipal.

Una vez procesado el documento, los datos almacenados podrán utilizarse para alimentar la tabla principal tbl_obras, desde donde se gestionará el seguimiento completo de la obra.
